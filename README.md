  <h3>1. 基本方針</h3>
  <p>本アプリ「Setlog(セットログ)」(以下「本アプリ」)は、利用者のプライバシーを最大限尊重します。本アプリで入力・記録されたデータ(ワークアウト記録、種目、ルーティン等)は、すべて利用者の端末内にのみ保存されます。</p>
